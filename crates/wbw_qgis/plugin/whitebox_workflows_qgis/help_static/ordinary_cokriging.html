Ordinary CoKriging Interpolation

Performs multivariate spatial interpolation using auxiliary variables to improve primary variable predictions. Ideal when primary data are sparse but correlated secondary data are abundant.

geostatistics cokriging multivariate raster interpolation

Parameters

NameDescriptionRequiredDefault
primary_pointsVector layer with primary variable training pointsRequiredprimary.gpkg
primary_fieldField containing primary variable valuesRequiredvalue
auxiliary_inputsComma-separated list of auxiliary variable file pathsRequiredaux1.gpkg,aux2.tif
auxiliary_fieldsComma-separated field names for auxiliary variables (empty for rasters)Requiredvalue,
template_rasterRaster template defining output grid and CRSRequiredtemplate.tif
outputOutput kriged raster pathRequiredcokriged.tif
output_varianceOptional output kriging variance raster pathOptional—
neighborhood_sizeNumber of nearest neighbors for local cokriging (default: all)Optional-1

Examples

Basic cokriging with one auxiliary variable

wbe.ordinary_cokriging(auxiliary_fields='value,', auxiliary_inputs='aux1.gpkg,aux2.tif', neighborhood_size=-1, output='cokriged.tif', output_variance='', primary_field='value', primary_points='primary.gpkg', template_raster='template.tif')
