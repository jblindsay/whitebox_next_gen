Split Lines At Intersections

Splits a line network wherever line segments intersect, including self-intersections.

vector network preprocessing split

Parameters

NameDescriptionRequiredDefault
inputInput line layer to split at intersections.Requirednetwork_lines.shp
snap_toleranceOptional snapping tolerance used by split operations; defaults to machine epsilon.Optional2.220446049250313e-16

Examples

Splits all line features at line-line intersections within one network layer.

wbe.split_lines_at_intersections(input='network_lines.shp', output='network_split_at_intersections.shp', snap_tolerance=2.220446049250313e-16)
