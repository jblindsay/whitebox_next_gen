Georeference Raster From Control Points

Fits a transform from GCPs and warps a raster into georeferenced output.

raster projection georeferencing

Parameters

NameDescriptionRequiredDefault
inputInput raster path.Requiredinput.tif
control_pointsControl point CSV path with pixel_x,pixel_y,map_x,map_y (and accepted aliases).Requiredinput_gcps.csv
epsgTarget EPSG code.Required32618
transform_typeTransform type: affine, projective, polynomial, or thin_plate_spline.Optionalpolynomial
transform_orderPolynomial order when transform_type=polynomial.Optional1
resampleResampling method; default bilinear.Optionalbilinear
allow_auto_downgradeAllow automatic downgrade to a simpler transform when GCPs are sparse.OptionalFalse
outputOutput georeferenced raster path.Optionalgeoreferenced.tif
reportOutput report JSON path.Optionalgeoreferenced_report.json

Examples

Georeference a scanned raster from control points using bilinear resampling.

wbe.georeference_raster_from_control_points(allow_auto_downgrade=False, control_points='input_gcps.csv', epsg=32618, input='input.tif', output='georeferenced.tif', report='georeferenced_report.json', resample='bilinear', transform_order=1, transform_type='polynomial')
