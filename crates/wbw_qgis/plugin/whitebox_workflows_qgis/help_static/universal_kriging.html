Universal Kriging

Performs universal kriging interpolation when the spatial phenomenon has a systematic trend or drift. Combines kriging with polynomial trend surface estimation.

raster gis kriging interpolation spatial-stats

Parameters

NameDescriptionRequiredDefault
pointsInput points vector layer.Requiredpoints.geojson
value_fieldNumeric attribute field containing values to interpolate.Optionalvalue
trend_orderPolynomial trend order (1=linear, 2=quadratic); defaults to 1.Optional1
cell_sizeOutput cell size in map units (required if no base_raster).Optional1.0
base_rasterOptional base raster controlling output geometry.Optional—
outputOptional output raster path.Optional—

Examples

Interpolates point values to a raster using universal kriging with trend.

wbe.universal_kriging(cell_size=1.0, output='universal_kriging.tif', points='points.geojson', trend_order=1, value_field='value')
