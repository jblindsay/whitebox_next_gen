Estimate Variogram

Computes an empirical semivariogram from point observations to characterize spatial correlation structure. Essential first step in geostatistical workflow.

geostatistics kriging spatial-statistics

Parameters

NameDescriptionRequiredDefault
inputInput vector points layer with valuesRequiredpoints.gpkg
fieldField containing measurement valuesRequiredvalue
lag_distanceLag bin size in map units (default: 100.0)Optional100.0
lag_toleranceTolerance for lag binning (default: 50.0)Optional50.0
max_lag_countMaximum number of lags to retain (default: 20)Optional20

Examples

Estimate variogram from point observations

wbe.estimate_variogram(field='value', input='points.gpkg', lag_distance=100.0, lag_tolerance=50.0, max_lag_count=20)
