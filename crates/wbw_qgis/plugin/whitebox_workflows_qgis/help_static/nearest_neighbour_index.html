Nearest Neighbour Index

Evaluate point-pattern clustering or dispersion using the Nearest Neighbour Index (NNI).

vector spatial-statistics point-pattern

Parameters

NameDescriptionRequiredDefault
inputInput point vector layer.Requiredpoints.gpkg
study_area_modeStudy area mode: hull, envelope, polygon_layer.Optionalhull
study_area_polygonPolygon layer used when study_area_mode=polygon_layer.Optional—
output_jsonOptional JSON report output path.Optional—
output_htmlOptional HTML report output path.Optional—
output_csvOptional CSV summary output path.Optional—

Examples

Computes NNI and writes a JSON report.

wbe.nearest_neighbour_index(input='points.gpkg', output_json='nni_report.json', study_area_mode='hull')
