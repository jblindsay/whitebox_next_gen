Getis-Ord Gi* - Raster Output

Computes Gi* hotspot/coldspot classifications from points and outputs raster (-1=cold, 0=NS, 1=hot). For hotspot-based analysis.\

raster spatial-statistics hotspot coldspot surface

Parameters

NameDescriptionRequiredDefault
inputInput vector layer with observation points.Requiredinput.gpkg
fieldNumeric attribute field to analyze.Requiredvalue
weights_modeNeighborhood mode: queen, rook, k_nearest, distance_band.Optionalk_nearest
kk value for k_nearest mode (default 8).Optional8
distanceDistance threshold for distance_band mode.Optional—
row_standardizeApply row standardization to weights (default true).OptionalTrue
island_policyIsland handling: drop_with_warning, keep_zero_weight, error.Optionaldrop_with_warning
alphaSignificance threshold in [0, 1]; default 0.05.Optional0.05
cell_sizeOutput raster cell size (optional; auto-computed from extent).Optional—
outputOutput raster path (classification: -1=Cold, 0=NS, 1=Hot).Required—

Examples

Computes Gi* and interpolates hotspots/coldspots to a raster surface.

wbe.getis_ord_gi_star_raster(alpha=0.05, field='value', input='input.gpkg', island_policy='drop_with_warning', k=8, output='hotspots_surface.tif', row_standardize=True, weights_mode='k_nearest')
