Snap Events To Routes

Snaps event points to route lines and reports route measure/offset diagnostics.

vector network linear-referencing event-snapping

Parameters

NameDescriptionRequiredDefault
routesInput route line layer.Requiredroutes.gpkg
eventsInput event point layer.Requiredevents.gpkg
max_offset_distanceOptional maximum route offset distance in map units.Optional—
outputOutput snapped event point vector path.Required—

Examples

Projects events to routes and emits route id, measure, and offset attributes.

wbe.snap_events_to_routes(events='events.gpkg', max_offset_distance=50.0, output='events_snapped_to_routes.gpkg', routes='routes.gpkg')
