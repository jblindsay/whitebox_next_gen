FFT Random Field

Creates a spatially-autocorrelated random field using FFT spectral synthesis.

raster simulation fft

Parameters

NameDescriptionRequiredDefault
base_rasterBase raster path used for grid geometry.Required—
rangeApproximate correlation range in map units. Default: 1.0.Optional1.0
outputOptional output raster path.Optional—

Examples

Create an autocorrelated random raster using FFT spectral filtering.

wbe.fft_random_field(base_raster='dem.tif', output='fft_random_field.tif', range=250.0)
