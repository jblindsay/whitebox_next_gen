OBIA Pipeline Basic

Executes complete end-to-end OBIA workflow: segmentation (SLIC/Graph), small-region merge, spectral/shape feature extraction, and random-forest classification in single operation.

remote_sensing obia workflow open-core

Parameters

NameDescriptionRequiredDefault
inputsArray of single-band input rasters for segmentation/features.Required—
trainingTraining CSV with segment_id and class columns.Required—
output_prefixOutput path prefix for generated artifacts.Required—
segment_methodSegmentation method: slic or graph (default slic).Optionalslic
min_sizeMinimum segment size for merge cleanup (default 10).Optional10
class_fieldTraining class field name (default class).Optionalclass

Examples

Run the baseline open-core OBIA pipeline.

wbe.obia_pipeline_basic(inputs=['red.tif', 'green.tif', 'nir.tif'], output_prefix='results/field01', segment_method='slic', training='training_segments.csv')
