Simple Kriging

Performs simple kriging interpolation when the mean of the spatial field is known a priori. This method reduces kriging variance and provides more stable predictions when the mean is reliably known from external information.

raster gis kriging interpolation spatial-stats

Parameters

NameDescriptionRequiredDefault
pointsInput points vector layer.Requiredpoints.geojson
value_fieldNumeric attribute field containing values to interpolate.Optionalvalue
known_meanKnown mean of the spatial field.Optional0.0
cell_sizeOutput cell size in map units (required if no base_raster).Optional1.0
base_rasterOptional base raster controlling output geometry.Optional—
outputOptional output raster path.Optional—

Examples

Interpolates point values to a raster using simple kriging.

wbe.simple_kriging(cell_size=1.0, known_mean=0.0, output='simple_kriging.tif', points='points.geojson', value_field='value')
