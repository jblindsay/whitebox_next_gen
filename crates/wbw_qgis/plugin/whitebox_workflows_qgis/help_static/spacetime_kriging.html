Space-Time Kriging

Performs space-time kriging interpolation to jointly model spatial and temporal variation in point measurements. Ideal for spatially-distributed time series data.

raster gis kriging interpolation spatial-stats

Parameters

NameDescriptionRequiredDefault
pointsInput points vector layer.Requiredpoints.geojson
value_fieldNumeric attribute field containing values to interpolate.Optionalvalue
time_fieldTemporal attribute field containing time values.Optionaltime
cell_sizeOutput cell size in map units (required if no base_raster).Optional1.0
base_rasterOptional base raster controlling output geometry.Optional—
outputOptional output raster path.Optional—

Examples

Interpolates point values to a raster using space-time kriging.

wbe.spacetime_kriging(cell_size=1.0, output='spacetime_kriging.tif', points='points.geojson', time_field='time', value_field='value')
