Getis-Ord Gi / Gi*

Compute local Gi/Gi* hot-spot statistics and significance classes for vector attributes.

vector spatial-statistics hotspot coldspot permutation-testing

Examples

Computes Gi* with asymptotic inference.

wbe.getis_ord_gi_star(alpha=0.05, field='value', inference='asymptotic', input='input.gpkg', island_policy='drop_with_warning', k=8, num_simulations=999, output='gi_star_output.gpkg', row_standardize=True, variant='gi_star', weights_mode='k_nearest')

Computes Gi* with permutation-based inference (999 simulations).

wbe.getis_ord_gi_star(alpha=0.05, field='value', inference='permutation', input='input.gpkg', island_policy='drop_with_warning', k=8, num_simulations=999, output='gi_star_output_perm.gpkg', row_standardize=True, variant='gi_star', weights_mode='k_nearest')
