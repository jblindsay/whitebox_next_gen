Quadrat Count Test

Run quadrat-based chi-square randomness tests for point-pattern analysis.

vector spatial-statistics point-pattern

Parameters

NameDescriptionRequiredDefault
inputInput point vector layer.Requiredpoints.gpkg
grid_modeGrid mode: rows_cols or cell_size.Optionalrows_cols
rowsRows when grid_mode=rows_cols.Optional10
colsCols when grid_mode=rows_cols.Optional10
cell_sizeCell size when grid_mode=cell_size.Optional—
study_area_modeStudy area mode: hull, envelope, polygon_layer.Optionalhull
study_area_polygonPolygon layer used when study_area_mode=polygon_layer.Optional—
output_gridOptional quadrat polygon grid output path.Optional—
output_jsonOptional JSON report output path.Optional—
output_htmlOptional HTML report output path.Optional—
output_csvOptional CSV summary output path.Optional—

Examples

Runs quadrat count test and writes JSON summary.

wbe.quadrat_count_test(cols=10, grid_mode='rows_cols', input='points.gpkg', output_json='quadrat_report.json', rows=10, study_area_mode='hull')
