Inhomogeneous Intensity - Raster Output

Computes kernel density estimation (KDE) surface visualizing spatial point intensity. Reveals hotspots and coldspots beyond simple density.

raster point-pattern kde

NameDescriptionRequiredDefault
inputInput point vector layer.Requiredinput.gpkg

outputOutput intensity raster.Required—

Examples

Compute kernel density estimation surface.

wbe.inhomogeneous_intensity_raster(input='input.gpkg', kernel='gaussian', output='intensity.tif')
