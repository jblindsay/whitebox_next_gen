Wisart Iterative Clustering

Wishart iterative clustering performs unsupervised classification of SAR polarimetric data by iteratively refining cluster centers using the complex Wishart statistical distance metric, which measures similarity in multivariate polarimetric probability distributions. The algorithm initializes from H/α decomposition zones (providing 9 seed clusters with known physical interpretation), then enters an expectation-maximization-like loop: (1) compute Wishart distance from each pixel's estimated coherency matrix to each cluster prototype; (2) reassign pixels to closest cluster; (3) update cluster prototypes by averaging assigned pixel matrices; (4) iterate until convergence (pixel reassignment rate
