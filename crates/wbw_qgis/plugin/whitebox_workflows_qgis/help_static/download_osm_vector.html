Download OSM Vector

Downloads OpenStreetMap features from the Overpass API for a bounding box and writes the result as a vector layer.

vector openstreetmap osm download overpass online

Examples

Downloads OSM road network for a small bounding box and writes to GeoJSON.

wbe.download_osm_vector(cache_ttl_hours=24, chunk_large_aoi=True, chunk_max_area_deg2=4.0, chunk_parallel_requests=1, clip_to_extent=True, east=-80.4, filter_preset='roads', include_key_values='', include_lines=True, include_points=True, include_polygons=True, include_tags='', input_extent_epsg=4326, max_chunk_count=64, max_elements=50000, north=43.5, output='osm_roads.geojson', overpass_profile='main', overpass_url='https://overpass-api.de/api/interpreter', south=43.4, split_output_by_geometry=False, timeout_seconds=25, west=-80.5)
