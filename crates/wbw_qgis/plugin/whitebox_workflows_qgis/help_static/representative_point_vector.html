Representative Point Vector

Generates an interior point guaranteed to lie within or on each geometry using pole-of-inaccessibility, ideal for label placement in concave polygons.

vector geometry representative-point point-on-surface

Parameters

NameDescriptionRequiredDefault
inputInput vector layer.Requiredinput_polygons.gpkg
outputOutput point vector path.Required—

Examples

Creates an interior representative point for each polygon feature.

wbe.representative_point_vector(input='input_polygons.gpkg', output='representative_points.gpkg')
