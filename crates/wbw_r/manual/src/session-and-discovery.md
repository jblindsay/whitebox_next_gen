# Session and Discovery

Core concepts:
- Session creation via `wbw_session(...)`.
- Tool visibility checks via `wbw_tool_ids(...)` and `wbw_has_tool(...)`.
- Search and metadata discovery via `wbw_search_tools(...)` and `wbw_describe_tool(...)`.
- Category-based discovery functions.

Planned examples:
- Category summary and per-category listings.
- Discovery patterns for workflow validation.
