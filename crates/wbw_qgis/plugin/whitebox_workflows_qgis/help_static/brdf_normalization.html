BRDF Normalization

Single-scene BRDF normalization using C-correction or Minnaert approach with DEM slope/aspect geometry.

remote-sensing brdf normalization radiometric terrain

Parameters

NameDescriptionRequiredDefault
input_rasterInput reflectance raster to normalize (single band or multi-band stack).Requiredinput.tif
input_demDEM raster co-registered with the input (same grid or will be resampled to match).Requireddem.tif
solar_zenith_degSolar zenith angle in degrees at acquisition time.Required40.0
solar_azimuth_degSolar azimuth angle in degrees (clockwise from north) at acquisition time.Required165.0
methodCorrection method: c_correction (default) or minnaert.Optionalc_correction
minnaert_kMinnaert k coefficient in [0,1] (default 0.5). Only used when method=minnaert.Optional0.5
z_factorVertical exaggeration for DEM slope/aspect computation (default 1.0).Optional1.0
output_prefixPrefix for output files (default brdf_normalized).Optionalbrdf_normalized
outputOutput path for normalized raster (overrides output_prefix for single-output mode).Optional—

Examples

Apply C-correction BRDF normalization to a reflectance band.

wbe.brdf_normalization(input_dem='dem.tif', input_raster='input.tif', method='c_correction', minnaert_k=0.5, output_prefix='brdf_normalized', solar_azimuth_deg=165.0, solar_zenith_deg=40.0, z_factor=1.0)
