Directional Variogram Analysis

Computes variograms in multiple directions to detect spatial anisotropy. Reveals directional continuity patterns essential for realistic kriging.

geostatistics variography anisotropy directional visualization

Parameters

NameDescriptionRequiredDefault
inputPoint layer with measurement valuesRequiredsamples.gpkg
fieldField containing measurement valuesRequiredvalue
directionsComma-separated list of azimuths (0-180°) to analyzeRequired0,45,90,135
toleranceDirection tolerance in degrees (default 22.5)Optional22.5
max_distanceMaximum lag distance to compute (default 1000)Optional1000.0
bin_sizeSize of lag bins (default 50)Optional50.0
output_jsonOutput file path for directional variogram resultsRequireddirectional_vgram.json
output_htmlOptional output path for HTML rose diagram visualizationOptionaldirectional_vgram_rose.html

Examples

Compute directional variograms at 4 azimuths (0°, 45°, 90°, 135°) with rose diagram

wbe.directional_variogram(bin_size=50.0, directions='0,45,90,135', field='value', input='samples.gpkg', max_distance=1000.0, output_html='directional_vgram_rose.html', output_json='directional_vgram.json', tolerance=22.5)
