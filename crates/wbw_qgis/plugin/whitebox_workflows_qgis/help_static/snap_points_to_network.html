Snap Points To Network

Snaps input point features to the nearest location along a network line layer.

vector network snapping preprocessing

Parameters

NameDescriptionRequiredDefault
pointsInput point layer to be snapped.Requiredpoints.gpkg
networkInput network line layer.Requirednetwork.gpkg
max_snap_distanceOptional maximum snapping distance in map units.Optional—
keep_unsnappedIf true (default), retains points that cannot be snapped within tolerance.OptionalTrue
outputOutput snapped point vector path.Required—

Examples

Snaps point features to nearest network edges and writes snap diagnostics fields.

wbe.snap_points_to_network(keep_unsnapped=True, max_snap_distance=50.0, network='network.gpkg', output='snapped_points.gpkg', points='points.gpkg')
